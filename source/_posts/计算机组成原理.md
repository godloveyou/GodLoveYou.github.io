## 无符号数
1.寄存器的位数  反映了无符号数的表示范围

## 有符号数
1.机器数：保存在计算机中的数据， 符号数字化的数
2.真值：数据真实的值，带符号的数


## 原码转补码
原码除符号位 按位取反, 末尾加1
```
原码: 1010 0101
补码: 1101 1011
```


## 补码转原码
将补码 除符号位 按位取反,末尾加1


```
补码: 1101 1011
原码: 1010 0101

```

## 已知 y(补) 求 (-y)补？
将y(补) 连同 符号位在内，每位取反,末位加1,即得(-y)补


为什么要引入移码？
补码无法直观的比较大小，移码一般用于表示浮点数的阶码
